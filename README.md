# Foldora — AI File Organizer for Windows & macOS

![Foldora demo](https://github.com/foldora-ai/foldora-ai.github.io/blob/main/src/assets/foldora-demo.gif)

Foldora is an AI-powered file organizer that automatically cleans messy folders such as Downloads, Desktop, and Documents.

Instead of manually sorting files, Foldora analyzes filenames and file content locally and organizes everything into structured folders automatically.

Foldora helps users organize downloads folders, rename messy files, clean desktop clutter, and improve file management workflows using local AI.

---

# 🚀 Features

- Automatic file organization
- AI-powered file categorization
- Smart file renaming
- Downloads folder cleanup
- Desktop clutter cleanup
- Offline AI processing
- Privacy-first local organization
- No cloud uploads
- Windows & macOS support

---

# 📂 What Foldora Does

Over time folders such as Downloads and Desktop become chaotic with:

- PDFs
- screenshots
- ZIP archives
- installers
- invoices
- work documents
- random files

Foldora automatically scans folders and creates a clean organized structure for your files.

---

# 🧠 How It Works

1. Select a folder  
2. Foldora analyzes files locally using AI  
3. Files are automatically grouped into structured folders  
4. Messy filenames can be renamed into readable names  

Everything happens locally on your device.

---
